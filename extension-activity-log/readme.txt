== Description ==
Tracks plugin activation/deactivation, usage stats, and admin alerts with CSV export and chart visualizations. Auto-cleans unused plugins and alerts admin.

== Installation ==
1. Upload `extension-activity-log` to the `/wp-content/plugins/` directory.
2. Activate through the ‘Plugins’ menu.
3. View logs in the new admin menu page.
